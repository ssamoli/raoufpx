⚡ AUTOMATE WITH FIREBASE

When you save contacts above, the data is stored on your Railway server (friendWhatsapp, friendName, messageScheduledAt on each referral code). Hook this to Firebase to auto-send the message below.

01
Firebase Console → New project → Enable Firestore Database + Cloud Functions
02
In your Railway server.js, inside /save-referral-contacts after saveCodes(), add:
db.collection('referral_queue').add({ code, whatsapp: friendWhatsapp, name: friendName, parentCode, createdAt })
03
Create a Cloud Function on firestore.document('referral_queue/{id}').onCreate — send this message via Twilio or Wati.io:
Hi [NAME]! I just booked a portrait session with RAOUF px in Dubai and got you an exclusive code. You can unlock your private session with this code: [REFERRAL CODE] Visit: https://raoufpx.com/secret.html and enter the code to unlock the offer. Expires in 48 hours!
04
After sending, update the Firestore doc: status: "sent" to prevent re-triggering.
05
For UAE numbers without WhatsApp Business API, use Twilio Sandbox (free to test) or Wati.io — both support template messages to +971 numbers out of the box.
